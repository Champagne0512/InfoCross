InfoCross 产品需求与设计规范 v4.0
项目名称	InfoCross (校园跨学科信息聚合平台)
设计代号	Morandi Flow (莫兰迪流光)
核心风格	全屏铺满 (Full Width)、高级灰调 (Muted Tones)、杂志化排版 (Editorial)
核心目标	打造一个既有学术深度，又兼具生活美学的全屏数字化校园空间。
1. 🎨 视觉设计规范 (Visual Design System)
1.1 色彩体系：莫兰迪高级灰 (The Morandi Palette)
摒弃高饱和度的荧光色，改用低饱和度、带有灰调的高级色系。这会让界面看起来非常耐看、有质感，且不仅限于黑白。
背景层 (Canvas):
Soft Cream (柔霜白): #FDFCF8 (代替纯白，一种极淡的暖调米白，更有纸张感)。
Mist Grey (迷雾灰): #F2F4F6 (用于侧边栏或分割区块)。
功能色 (莫兰迪色谱 - 用于卡片背景、标签、按钮):
🌿 Sage Green (鼠尾草绿) - [运动/生活]: #A6B9A8
🌊 Hazy Blue (雾霾蓝) - [科研/学术]: #93A8AC
🧱 Dusty Clay (脏粉/陶土) - [娱乐/社交]: #D9A69F
🔮 Muted Lavender (灰紫) - [AI/破壁]: #B4A8BF (AI 的专属色，低调神秘)
文字色 (Typography Colors):
Charcoal (炭灰): #374151 (主标题，非纯黑，柔和)。
Slate (板岩灰): #6B7280 (正文，清晰易读)。
1.2 字体系统 (Typography)
要求高级、现代的无衬线字体 (Sans-Serif)。
主字体 (Headings & Body): Manrope 或 Outfit。
特点: 几何感强，但边角有微小的圆润处理，非常现代且优雅。
功能字体 (Data/Tags): DM Mono。
特点: 这种等宽字体比 JetBrains Mono 更“人文”一点，不像代码那么生硬，适合莫兰迪风格。
1.3 布局原则：全屏流体 (Fluid Full-Width)
拒绝中栏定宽： 不要让内容只挤在屏幕中间 1200px 的区域。
铺满屏幕： 内容区域宽度设为 100%，左右留出 32px 或 48px 的呼吸边距 (padding-x)。
网格系统： 使用 Grid Layout (CSS Grid)，将屏幕横向分为 12 列，根据屏幕宽度自动拉伸。
2. 页面详细布局 (Page Layout Specs)
2.1 全局导航 (Global Nav)
形式： 左侧固定侧边栏 (Sticky Left Sidebar)。
宽度： 固定 240px，高度 100vh（铺满垂直屏幕）。
样式： 背景色为 #FDFCF8，右侧有一条极细的分割线。
内容：
顶部：Logo (InfoCross)。
中部：巨大的菜单项（首页、组队、发布、我的），选中时背景变为极淡的莫兰迪色块。
底部：用户头像 + 设置。
2.2 首页：The Hub (全景控制台)
布局逻辑： 除去左侧导航，右侧内容区采用 4列流体网格 (Fluid Grid)。内容铺满，不再留白。
区域 A：顶部问候与数据 (Header) - [跨 4 列]
一个全宽的横幅。背景是极淡的渐变灰。
左侧显示：“早安，同学。”
右侧横向排列三个磨砂玻璃数据卡片：[今日入库: 42] [破壁指数: 89%] [待办事项: 3]。
区域 B：AI 破壁精选 (Featured) - [左侧 3 列]
这里展示 AI 强推的一条内容。
样式： 巨大的卡片，使用 灰紫色 (Muted Lavender) 背景。
内容包含：巨大的标题、AI 生成的推荐理由（“✨ 既然你关注Python，不妨看看这个艺术展”）、以及一张充满艺术感的配图。
区域 C：全校动态瀑布流 (Main Feed) - [左侧 3 列]
位于区域 B 下方。
采用 3 列或 4 列并排 的瀑布流卡片。卡片没有边框，只有柔和的阴影。
卡片配色： 根据活动类型，卡片底部有一条细细的莫兰迪色条（绿/蓝/粉）。
区域 D：辅助信息栏 (Widget) - [最右 1 列]
注意：这一列虽然在右边，但视觉上要和左边融合，不要有明显的分割线。
组件 1： 本周热词（词云图，使用莫兰迪色块拼接）。
组件 2： 今日摘要 (TL;DR)（一条长长的便签纸风格，列出今日重点）。
2.3 组队页：The Collab (协作拼贴墙)
布局逻辑： 模仿“便利贴墙”的感觉，视觉极其丰富，铺满整个屏幕。
顶部： 筛选器（全宽）。
一排圆形的色块按钮：[💻 代码类] [🎨 设计类] [📈 商科类]。
主体： Masonry 布局 (砖石流)。
卡片不再是整齐的矩形，可以有长有方。
背景色： 卡片背景直接使用极淡的莫兰迪色（淡绿、淡蓝、淡粉），像一张张真实的便利贴贴在墙上。
技能缺口： 在便利贴上画几个虚线圆圈，代表缺人。
交互： 鼠标悬停时，便利贴的一角会有微微“揭起”的 3D 效果。
2.4 发布页：The Studio (沉浸式创作台)
布局逻辑： 左右 50:50 分屏，铺满屏幕，无边距。
左半屏 (编辑器)：
背景色：#FDFCF8 (米白)。
内容：极简的表单。输入框没有边框，只有底部的线条。
AI 拖拽区： 一个巨大的虚线框，占据屏幕上方。
右半屏 (实时预览)：
背景色：迷雾灰 (#F2F4F6)。
内容：
屏幕中央悬浮着当前正在编辑的“卡片预览图”。
底部有一个浮动的AI 日志条（半透明磨砂黑），实时显示 AI 正在分析的进度。
2.5 个人页：The Profile (数据画卷)
布局逻辑： 模块化拼接，像一本打开的杂志。
板块 1：数字 ID 卡 (Top Left) - [占 1/3 宽]
竖向的长卡片。包含大头像、姓名、学院。
底部有一个莫兰迪色的二维码。
板块 2：能力雷达 (Top Right) - [占 2/3 宽]
一个巨大的矩形区域。
左边显示雷达图（五维数据），右边显示文字版的“破壁分析报告”。
板块 3：成就勋章墙 (Bottom) - [全宽]
横向铺满屏幕。
展示一排排矢量勋章。点亮的勋章使用莫兰迪配色，未点亮的显示灰色线条。
3. UI 组件细节规范 (Component Details)
3.1 莫兰迪卡片 (The Morandi Card)
不要用普通的白底黑字卡片。
背景： 使用极淡的莫兰迪色背景 (例如 bg-blue-50 但饱和度更低)。
圆角： 24px (非常圆润)。
无边框模式： 完全去掉边框，利用色彩区分卡片和背景。
内容布局：
图片在上，占 60% 高度。
文字在下，标题使用 Manrope 粗体。
标签悬浮在图片右上角，使用磨砂玻璃效果。
3.2 按钮 (The Soft Button)
主按钮： 背景色为 炭灰 (#374151)，文字白色。圆角 12px。
次要按钮： 背景色为 淡莫兰迪色（如淡绿），文字为深绿。
动效： 悬停时，按钮不会变色，而是整体放大 2%，并增加投影。
3.3 AI 标签 (The AI Tag)
样式： 不要是实心色块。
设计： 左侧是一个小星星图标 ✨，右侧是文字。
颜色： 统一使用 灰紫色 (#B4A8BF) 的文字，背景透明，边框为淡紫色细线。
4. 给开发者的 CSS/Tailwind 提示
为了实现“全屏铺满”和“莫兰迪色”，请在 Tailwind 配置中加入以下自定义：
code
JavaScript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      // 莫兰迪色盘
      colors: {
        cream: '#FDFCF8',
        mist: '#F2F4F6',
        charcoal: '#374151',
        morandi: {
          green: '#A6B9A8',
          blue: '#93A8AC',
          clay: '#D9A69F',
          lavender: '#B4A8BF',
        }
      },
      // 字体
      fontFamily: {
        sans: ['Manrope', 'Outfit', 'sans-serif'],
        mono: ['DM Mono', 'monospace'],
      },
      // 布局实用类
      gridTemplateColumns: {
        'dashboard': '240px 1fr', // 左侧导航固定，右侧自适应
      }
    }
  }
}
总结
这套 "Morandi Flow" (莫兰迪流光) 方案完美满足了你的所有新要求：
色彩丰富且高级： 告别黑白，引入了整套低饱和度的高级彩色。
字体高级： Manrope + DM Mono 的组合非常显贵。
全屏铺满： 利用 100% 宽度和网格布局，信息量大，不浪费大屏幕空间。
布局清晰： 导航在左，内容在右，分工明确。
这是一个非常有艺术感、且完全符合现代 Web 审美的设计方案。
